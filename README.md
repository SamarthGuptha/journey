# journey

A browser-based 3D visualization tool that turns your JavaScript/TypeScript code into an interactive, cyberpunk-style graph!

NOTE: Because this project uses ES Modules (to load Three.js), modern browsers (Chrome/Firefox) may block it if you simply double-click the index.html file due to CORS security policies.

## Guided Mode (Default)

Scroll Wheel: Automatically fly the camera to the next node in the sequence.

Left Click: Select a specific node (from the 3D view or the sidebar list!) to focus on it.

Hover: View the code snippet and inferred data payload.

## Free Roam Mode
Click the "Enable Free Roam" button at the bottom. 

Left Click + Drag: Rotate camera.

Right Click + Drag: Pan camera. 

Scroll Wheel: Zoom in/out.
